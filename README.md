# Laser Prison

Technical test developed in Unity.

Objective: Survive as long as possible while avoiding lasers.

## Controls

| Action          | Input             |
| --------------- | ----------------- |
| Move            | WASD / Left Stick |
| Invulnerability | Space / Button South|
